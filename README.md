# OZE Poznań

Lokalny portal o odnawialnych źródłach energii w Poznaniu i Wielkopolsce.

## Stack
- Astro 4.x (static output)
- Vanilla CSS (Eco Green — nature/sustainability theme)
- Cloudflare Pages deployment

## Uruchomienie
```bash
npm install
npm run dev
```

## Deploy
Cloudflare Pages: build `npm run build`, publish dir `dist`
